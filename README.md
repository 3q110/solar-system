# ☀️ 太阳系天体运动模型

基于真实天文数据的开普勒轨道太阳系模拟器，纯前端实现（单个 HTML 文件，无需服务器）。

## ✨ 特性

- **真实天文数据** — 行星轨道参数来源 NASA JPL HORIZONS / JPL Solar System Dynamics
- **完整太阳系** — 8 大行星 + 太阳 + 月球 + 主要卫星（木卫/土卫/天王卫/海卫）+ 2 颗彗星
- **小行星带 & 柯伊伯带 & 奥尔特云** — 三层外围结构示意
- **开普勒方程求解** — 真实的椭圆轨道、偏心率、轨道倾角、真近点角计算
- **3D 光照渲染** — 行星本体带光照渐变，土星/天王星带环系统
- **玻璃质感 UI** — 毛玻璃面板、深空配色、响应式布局

## 🎮 操作说明

### 鼠标操作

| 操作 | 功能 |
|------|------|
| **滚轮** | 缩放视图（以鼠标位置为中心） |
| **拖拽** | 平移视图 |
| **单击天体** | 查看详细天文数据 |

### 键盘快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 暂停 / 继续 |
| `+` / `-` | 加速 / 减速模拟速度 |
| `←` `→` `↑` `↓` | 方向键平移视图 |
| `G` | 打开/关闭行星导航面板 |
| `R` | 重置视图到初始状态 |
| `S` | 截图保存为 PNG |

### 底部控制栏按钮

| 按钮 | 功能 |
|------|------|
| **速度滑块** | 调节时间流速（对数刻度：0.01x ~ 1000x） |
| **轨道** | 显示/隐藏行星轨道路径 |
| **标签** | 显示/隐藏天体名称标签 |
| **轨迹** | 开启/关闭行星运动轨迹记录 |
| **倾角** | 启用/禁用轨道倾角效果（Z轴） |
| **卫星** | 显示/隐藏各行星的卫星 |
| **暂停** | 暂停或继续模拟 |
| **重置** | 视图归位（居中太阳，缩放1x） |
| **聚焦地球 / 聚焦太阳** | 相机快速锁定目标天体 |
| **取消聚焦** | 解除锁定，回到自由视角 |
| **自转** | 开启自动环绕旋转（未聚焦时生效） |
| **🌍 导航** | 弹出行星快速导航列表 |
| **📷 截图** | 一键保存当前画面（自动隐藏 UI） |

## ⚙️ 技术细节

- **物理引擎**: 开普勒方程迭代求解（牛顿法，精度 1e-10）
- **坐标系统**: 日心黄道坐标系，AU 单位
- **渲染**: HTML5 Canvas 2D，60fps requestAnimationFrame
- **时间基准**: J2000.0 历元（2000-01-01）
- **速度基准**: 1x = 1 地球日/秒（地球公转需约 365 秒）

## 📝 使用提示

1. **入门推荐流程**: 打开后先点「聚焦地球」→ 调整速度到 10x → 观察月球绕地运动 → 切换「聚焦太阳」看全局
2. **观察外行星**: 点「聚焦木星」或「聚焦土星」，可以看到伽利略卫星和土星环
3. **长时间运行**: 开启「轨迹」功能，加速到 100x 以上，观察行星轨道形状
4. **截图分享**: 按 S 键或点「📷 截图」，可直接保存精美画面
